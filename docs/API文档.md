# API接口文档

## 1. 获取食堂状态
**接口**: `/api/canteen/status`  
**方法**: GET  
**功能**: 获取当前食堂整体状态  
**负责人**: 成员A

## 2. 获取窗口信息
**接口**: `/api/windows`  
**方法**: GET  
**功能**: 获取所有窗口的排队情况  
**负责人**: 成员A

## 3. 获取座位信息
**接口**: `/api/seats`  
**方法**: GET  
**功能**: 获取座位占用情况  
**负责人**: 成员A

## 4. 获取时间段分析
**接口**: `/api/analysis/time`  
**方法**: GET  
**功能**: 获取各时段统计数据  
**负责人**: 成员B

## 5. 获取推荐时间
**接口**: `/api/recommendation`  
**方法**: GET  
**功能**: 获取最佳就餐时间推荐  
**负责人**: 成员B
